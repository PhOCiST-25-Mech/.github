## Проекты (обязательны только первого потока, 20 баллов)

- постановка и связь с теорией — 4
- корректность модели/измерений — 5
- реализация/код/стенд — 9
- выводы — 2

- неделя 4: выбор темы
- неделя 9: промежуточный результат
- неделя 13–14: прогон презентации
